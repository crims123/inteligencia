Tenemos $n$ objectos numerados del $0$ a $n-1$, con un tama√±o $s_i$ y valor $v_i$ cada uno. Una soluci√≥n cualquiera del problema es el vector $X=[x_0, x_2, ..., x_i, ... x_{n-1}]$, en donde $x_i \in \{0,1\}$ indica si el objecto $i$ se incluye o no la mochila. Como la mochila tiene un volumen m√°ximo, $KS_{vol}$ el problema se formula de la siguiente manera:

$$ maximize \; X \;\; \sum_{i=0}^{n-1} x_i v_i$$$$subject \; to \;\; \sum_{i=0}^{n-1} x_i s_i \le KS_{vol}$$

El vector $S=[s_0, s_2, ..., s_i, ... s_{n-1}]$ representa los tama√±os de todos los objectos y el vector $V=[v_0, v_2, ..., v_i, ... v_{n-1}]$, los valores.

F√≠jate a continuaci√≥n c√≥mo:

  • generamos vectores $S$ y $V$ aleatorios
  • generamos soluciones aleatorias
  • verificamos si las soluciones generadas son v√°lidas

Ejercicio 2: Función de coste¶

El valor m√°ximo que tendr√≠amos si pudi√©ramos incluir todos los objetos en la mochila ser√≠a $V_{max}=\sum v_i$. El volumen de los objetos incluidos en una soluci√≥n $X$ es $S(X)=\sum s_i x_i$. El valor de todos los objetos incluidos en una soluci√≥n es $V(X)=\sum v_i x_i$

Crea una funci√≥n de coste de manera que valores altos de las combinaciones de objetos representen un coste bajo, siempre y cuando no excedan la capacidad de la mochila. Observa que nuestra implementaci√≥n de GA MINIMIZA un coste y nosotros queremos MAXIMIZAR un valor.

$$ KS\_cost(X)=\left\{ \begin{array}{ll} V_{max} - V(X) & si \;\; S(X) \le KS_{vol}\\ V_{max} & si \;\; S(X) \gt KS_{vol}\\ \end{array} \right. $$
In [68]:
def KS_cost(solution, object_values, object_volumes, KS_volume):

    # TU CODIGO AQUI
    Vmax=np.sum(object_values)
    Sx=np.sum(solution*object_volumes)
    Vx=np.sum(solution*object_values)
    if(Sx<=KS_volume):
        cost=Vmax-Vx
    else:
        cost=Vmax
        
    
    return cost

def KS_mutate(solution, mutation_prob, object_volumes, KS_volume):

    # TU CODIGO AQUI
    i1 = np.random.randint(len(solution))
    i2 = i1+1 if i1<len(solution)-1 else 0
    r = np.copy(solution) 
    if(np.random.random()<=mutation_prob):
        r[i1]=solution[i2]

    return r

def KS_cross_over(solution1, solution2):
    l = len(solution1)
    cstart = np.random.randint(l)
    cend   = np.random.randint(l-cstart)+cstart+1

    r = np.zeros(l).astype(int)
    r[:cstart]     = solution1[:cstart]
    r[cstart:cend] = solution2[cstart:cend]
    r[cend:]       = solution1[cend:]            
    return r
